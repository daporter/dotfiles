Hyprland Keybindings Reference

Window Focus

Key Action
Super+A Focus left
Super+E Focus up
Super+I Focus down
Super+H Focus right

Workspaces

Key Action
Super+1-9 Go to workspace 1-9
Super+K Next workspace
Super+U Previous workspace
Super+Tab Previous workspace
Super+Scroll Up Next workspace
Super+Scroll Down Previous workspace

Window Movement

Key Action
Super+Shift+A Swap left
Super+Shift+E Swap up
Super+Shift+I Swap down
Super+Shift+H Swap right
Super+LMB (drag) Move window

Move to Workspace

Key Action
Super+Shift+1-9 Move to workspace 1-9
Super+Shift+K Move to next workspace
Super+Shift+U Move to previous workspace

Applications

Key Action
Super+Space Launch wofi (app launcher)
Super+Ctrl+Space Launch 1Password Quick Access
Super+Return Launch kitty (terminal)
Super+B Launch Firefox
Super+Ctrl+D GoldenDict lookup (clipboard)
Super+Ctrl+E Launch Emacs (Client)
Super+Ctrl+S Screenshot window
Super+Ctrl+Shift+S Screenshot region

Window Resizing

Key Action
Super+Ctrl+A Resize left (-20px)
Super+Ctrl+E Resize up (-20px)
Super+Ctrl+I Resize down (+20px)
Super+Ctrl+H Resize right (+20px)
Super+RMB (drag) Resize window

Layout

Key Action
Super+F Toggle maximise
Super+Ctrl+F Toggle fullscreen
Super+Shift+Space Toggle floating
Super+Shift+LMB Toggle floating
Super+W Kill active window
Super+P Pin window

Scratchpad

Key Action
Super+S Toggle scratchpad
Super+Ctrl+T Launch scratchpad terminal

Hyprland

Key Action
Super+Shift+Escape Quit Hyprland

Mouse Buttons

  • LMB = Left Mouse Button (272)
  • RMB = Right Mouse Button (273)
